Introdução

Bootstrap é um framework desenvolvido em HTML/CSS/JS para acelerar a criação de um front-end de um(a) site/aplicação

Contando com a disposição de uma grid CSS com 12 colunas, que possibilitam exelente organização dos espaços, scripts para montagem de menus, formularios, Navbars, icones e uma imensidão de outras opções!

Criado pela equipe do Twitter, hoje e o framework HTML/CSS/JS mais popular, e com comunidade super-ativa, alem de ser super-intuitivo e fácil de usar.

Responsividade

O uso efetivo das grids é fundamental para um bom projeto com o Bootstrap e entender a logica por trás disso é simples

O sistema de grids possibilita a divisão em até 12 colunas de mesma largura

O uso de grids possibilita mudar o visual de um site ou sistema de maneira fácil e ráida, alterando apenas o valor da largura das colunas

As grids são nativamente responsivas, a definição as grids para diferentes tipos de resolução, são definidos atravez das classes:
XS para dispositivos muito pequenos
SM para dispositivos pequenos
MD para dispositivos médios
LG para dispositivos grandes
